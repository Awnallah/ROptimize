
The code was run with python 3.5 using jupyter notebook environment.
The code and data can be found in the following link:

https://github.com/Awnallah/ROptimize

Note: The mlrose code in the repository isn't the standard one. The version in the repository contains
minor recommended modification to make MIMIC faster by  David Sejin Park(github.com/parkds/mlrose).

Please also use the included training data in the repository as this is balanced data.
To run the Experiments, please run the jupyter notebooks (named according to experiments).

Finally, the data was resampled using RandomOverSample from imblearn.
